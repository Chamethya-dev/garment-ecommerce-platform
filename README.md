# Garment E-Commerce & Analytics Platform

A data-driven e-commerce platform tailored for a garment manufacturing business in Sri Lanka.

## 🚀 Features
- **Product Catalog:** Browse garments with variant filtering (Size, Color).
- **Shopping Cart & Checkout:** Seamless purchasing with district-based delivery fees.
- **Order Management:** Real-time order status tracking and exchange handling.
- **Inventory Tracking:** Automated, variant-level stock management with low-stock alerts.
- **Analytics Dashboard:** Business intelligence on revenue, sales trends, and customer behavior.
- **Data Science Integration:** Sales forecasting, RFM Customer Segmentation, and Product Recommendations.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend:** Python, Flask
- **Database:** MySQL
- **Data Science:** Pandas, Matplotlib, Scikit-Learn, Prophet
- **Deployment:** Render / Railway
